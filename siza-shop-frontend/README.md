# React Product Shop

React Technical assignment.

## Features

- Product listing
- Category filtering
- Search functionality
- User signup and login
- Add to cart with quantity management
- Cart and checkout

## Tech Stack

- React.js
- Axios
- React Router

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rishinck-hub/react-shop.git
```

### 2. Navigate to the project folder

```bash
cd siza-shop-frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open in browser

http://localhost:5173

## API Used

https://api.escuelajs.co/api/v1

## Notes

- Make sure Node.js is installed
- Internet connection is required for API calls
